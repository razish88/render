### Test the RESTAPI 

joke/invoke
```
curl --location --request POST 'http://localhost:8000/joke/invoke' \
    --header 'Content-Type: application/json' \
    --data-raw '{
        "input": {
            "topic": "cats"
        }
    }'

```
joke/batch

```
curl --location --request POST 'http://localhost:8000/joke/batch' \
    --header 'Content-Type: application/json' \
    --data-raw '{
        "inputs": [
            {"topic": "cats"},
            {"topic": "dogs"}
        ]
    }'
```
